# Android Device Fingerprint SDK

这是一个高性能、高安全性的安卓设备指纹采集 SDK，采用 **Kotlin + C++ (NDK)** 架构开发。支持采集硬件特征、传感器信息、电池状态，并具备底层的 Root 和模拟器检测能力。

## 🚀 核心功能

*   **硬件特征采集**：品牌、型号、CPU 架构/频率、屏幕 DPI、内存/存储容量等。
*   **环境检测**：
    *   **Root 检测**：结合 Java 层与 C++ 底层双重校验。
    *   **模拟器检测**：深度检测 QEMU 特征文件及底层系统属性。
    *   **风险检测**：VPN 状态、开发者模式、模拟定位（需权限）。
*   **安全交互**：采用自定义加密分隔符进行 JNI 数据传输，防止简单 Hook。
*   **现代架构**：全量支持 Kotlin，兼容 Java 21 & AGP 8.4.0。

## 📦 集成方式

### 方法一：通过 JitPack 远程依赖 (推荐)

在您的根目录 `build.gradle` (或 `settings.gradle`) 中添加 JitPack 仓库：

```gradle
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```

在您的 `app/build.gradle` 中添加依赖：

```gradle
dependencies {
    implementation 'com.github.wangshuo1779:fingerprint-sdk:v1.0.0'
}
```

### 方法二：直接导入源码模块

1. 将 `fingerprint-sdk` 文件夹复制到您的项目根目录。
2. 在 `settings.gradle` 中添加 `include ':fingerprint-sdk'`。
3. 在 `app/build.gradle` 中添加 `implementation project(':fingerprint-sdk')`。

## 🛠 使用步骤

### 1. 初始化 SDK
建议在 `Application.onCreate` 或主 Activity 中初始化：

```kotlin
import com.device.fingerprint.FingerprintSdk

// 初始化一次即可
FingerprintSdk.getInstance().init(context)
```

### 2. 采集数据
获取全量采集到的 `DeviceInfo` 数据模型：

```kotlin
// 1. 获取包含所有字段的数据对象
val info = FingerprintSdk.getInstance().collectDeviceInfo()

// 2. 转换为 JSON 格式方便上报
val jsonString = info.toJson().toString(4)

// 3. 或者直接获取唯一的指纹 ID (示例算法)
val fpId = FingerprintSdk.getInstance().getFingerprintId()
```

### 3. 混淆配置 (Proguard)
如果您的项目开启了代码混淆，请务必添加以下规则：

```pro
-keep class com.device.fingerprint.model.** { *; }
-keep class com.device.fingerprint.FingerprintSdk {
    private external <methods>;
    public *;
}
```
